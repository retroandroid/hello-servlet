# Java Servlet Assignment

This project is a simple Java Servlet that displays a hello message with my name.

## Tools used
- OS: Bazzite (Linux)
- Java: OpenJDK 21
- Apache Tomcat: 10.1.52
- Maven
- IntelliJ IDEA

## What I did
- Installed Apache Tomcat
- Created a Java Servlet
- Built the project using Maven
- Deployed the WAR file on Tomcat
- Tested the servlet in a web browser

## How to test
- Main page: http://localhost:8080/hello-servlet/
- Servlet: http://localhost:8080/hello-servlet/hello
